DISEÑO WEB

Este ejercicio formó parte del segundo obligatorio de la materia, para el cual había que realizar dos tareas. Una de ellas consistía en diseñar un prototipo para una aplicación de tema a elección. Ésta debía cumplir con determinadas características: alternar el modo de color entre claro y oscuro, poder editar la información del perfil, poder modificar la información relativa a las estadísticas que recoja la aplicación, etc.

Elegí diseñar una aplicación para leer libros, calificarlos y almacenar diferentes datos relacionados a las estadísticas de lectura. Para esto se diseñó el logotipo y la interfaz general.
